How to read this

  • Month-wise answers questions like “How has BTC usually done in August?” Each year shows that month’s return only when the month has finished.
  • Current year appears for a month only if that calendar month has already completed relative to today.
  • Year-wise uses full calendar years. The open year is shown as YTD through the latest completed month.
  • Returns are relative: ((end ÷ start) − 1) × 100. Positive is green, negative is red.
